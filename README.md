# what this one is about
This is a hobby project to create a simple and easy to deploy and scale IPAM tool. 
It is based on FastAPI and some database behind.

It is also using DevContainer to simplify the environment but it is required to be vscode with specifc addon enabled.
Local docker must be available as well.

To run the server please type:

uvicorn main:LeanIPGrid --reload
